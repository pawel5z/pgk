## Build instructions
From repo root dir: \
`$ make` \
Executable's name is `labN`, where `N` is the number of the assignment.

### Debug build
Compiles to executable with debug symbols. \
Delete any files without debug symbols. \
`$ make distclean` \
`$ make debug`
